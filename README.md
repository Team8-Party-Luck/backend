### 🔗 사이트
[Eatsring](https://eatsring.com)


### 🏠 소개
- Eatsring(잇츨링) 서비스는 <b>맛집탐방</b>과 <b>파티원 모집</b> 서비스입니다.
- 맛집탐방을 함께하고 싶은 파티원을 모집해주는 서비스입니다.
- 주요기능
  1. WebSocket을 이용한 실시간 채팅
